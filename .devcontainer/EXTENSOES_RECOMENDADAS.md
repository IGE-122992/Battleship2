# Extensões Recomendadas

Este documento apresenta algumas extensões recomendadas para facilitar o desenvolvimento, manutenção e documentação do projeto.

## Visual Studio Code

### Docker
Permite gerir containers, imagens, volumes e ficheiros Docker diretamente no VS Code.

Funcionalidades:
- Visualização de containers ativos
- Gestão de imagens Docker
- Execução de comandos Docker
- Acesso rápido a logs

### GitHub Actions
Facilita a visualização e acompanhamento dos workflows de CI/CD.

Funcionalidades:
- Visualizar execuções dos workflows
- Consultar logs
- Monitorizar estados de sucesso/falha

### Markdown All in One
Auxilia na edição de documentação Markdown.

Funcionalidades:
- Pré-visualização em tempo real
- Atalhos para formatação
- Geração automática de índices

### SQLTools
Permite ligação e consulta a bases de dados diretamente no editor.

Funcionalidades:
- Execução de queries SQL
- Exploração de tabelas
- Visualização de resultados

### MongoDB for VS Code
Facilita a interação com bases de dados MongoDB.

Funcionalidades:
- Navegação por coleções
- Consulta de documentos
- Execução de comandos MongoDB

## IntelliJ IDEA

### Docker Plugin
Integração com containers Docker e Docker Compose.

### GitHub Plugin
Integração com repositórios GitHub e pull requests.

### Database Tools and SQL
Ferramentas para gestão e consulta de bases de dados relacionais.

### Markdown Support
Suporte para edição e pré-visualização de ficheiros Markdown.

---

## Observação

Estas extensões não são obrigatórias para a execução do projeto, mas são recomendadas por melhorarem a produtividade, a manutenção do código e a gestão da infraestrutura utilizada.
